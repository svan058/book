## Repository Structure

### Root — The Book

| File | Description |
|------|-------------|

### `working/` — Supporting Material

Everything that isn't the book itself. Sorted by type.

#### Writing

- **`drafts/`** — Chapter work-in-progress, experimental sections, voice notes
- **`ch7-evidence/`** — Research and source queries for Chapter 7 (Reading the Room)
- **`book-overview-and-workbook-answers.md`** — Book structure overview + retreat workbook responses

#### Analysis (processed)

- **`analysis/`** — Gem mining, chapter evidence maps, structural assessments, incremental guidebook reviews
- **`reviews/`** — Manuscript feedback (Pascal, Claude reviews)

#### Research & References

- **`research/`** — Research notes (Perry, isolation/inner-peopling), published article reference
- **`source-materials/`** — Retreat guidebook + workbook PDFs (original reference docs)
- **`raw-inputs/`** — Unprocessed transcripts, raw source material not yet reviewed
- **`transcripts/`** — Processed transcripts (consciousness talk, Jiang Xueqin, etc.)

#### Agent / Tooling

- **`job-prompts/`** — Claude dispatch prompts
- **`runs/`** — Claude run logs (prompt + result per run)
- **`feature-requests/`** — Ideas for future book directions or features

#### Archive

- **`archive/`** — Superseded versions (`Peopling_Master_Manuscript.md`, old workbook fills, deprecated chapters). Do not edit.

## Writing Conventions

- **No em-dashes or en-dashes.** Use full stop, comma, colon, or sentence split instead.
- Voice skill: `stefan-book-voice` (essay register, conversational academic)
- Master manuscript is the single source of truth for "what's in the book"
- Individual chapter files at root are extracts for sharing/review, not canonical
